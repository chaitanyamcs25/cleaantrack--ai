CleanTrack AI is a smart waste reporting system that allows users to report garbage and uses AI-based analysis to classify waste and assign priority.

## 🚀 Live Website

https://chaitanyamcs25.github.io/cleantrack--ai/

## 💡 Features

* Report garbage using text input
* AI-based classification (type & priority)
* Reports dashboard
* Status tracking

## ⚙️ Tech Stack

* HTML, CSS, JavaScript
* Local Storage (for MVP)

## 🧠 How it works

User input → AI analysis (keyword-based) → classification → priority → report display

## 📌 Note

This is a prototype using lightweight AI simulation (keyword-based logic).
The system is designed to be extended with Google Gemini API for advanced AI capabilities.
